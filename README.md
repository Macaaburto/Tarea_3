# Tarea_3
Desarrollo desayuno

Elementos: Café, pan, palta, manjar, naranjas, azúcar, sal, aceite, taza, vaso, sartén.

Pasos de ejecución:

1.	Cortar por la mitad las naranjas para el jugo. 
2.	Preparar el jugo de naranja en un vaso (azúcar a gusto).
3.	Cortar por la mitad las paltas.
4.	Moler las paltas (la sal y el aceite es opcional).
5.	Calentar sartén para el batido del panqueque.
6.	Rellenar la porción de panqueques con manjar.
7.	Calentar el pan (a gusto personal).
8.	Preparar el pan caliente con palta.
9.	Servir agua hervida caliente en una taza.
10.	Preparar café (el azúcar es opcional).
11.	Servir tu rico desayuno.
